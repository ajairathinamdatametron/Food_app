🍔 Food Ordering App

A modern React Native food ordering application that allows users to browse food items, customize burgers, manage their cart, make secure payments, and track order history through a clean and user-friendly interface.

📱 Features
User Profile
View and edit profile information
Update profile picture
Save payment details
Food Menu
Browse multiple food categories
View food item details
Search for food items
Custom Burger
Create your own burger
Choose ingredients and customize your order
Cart Management
Add items to cart
Update item quantity
Remove items from cart
View total price
Payment
Secure payment integration using Razorpay
Payment success and failure screens
Order History
View previously placed orders
Display order date and time
Show order status
View ordered items with images
User Interface
Responsive design
Smooth navigation
Attractive food cards
Floating navigation buttons
🛠️ Technologies Used
React Native

React Navigation

Razorpay Payment Gateway

🚀 Getting Started
Prerequisites
Node.js
React Native CLI
Android Studio
Java JDK
Git
Installation

Clone the repository:

git clone https://github.com/ajairathinamdatametron/Food_app.git

Navigate to the project folder:

cd Food_app

Install dependencies:

npm install

Start the Metro Bundler:

npx react-native start

Run the Android application:

npx react-native run-android
